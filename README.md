# APIs 🚀

A structured collection of projects built while learning **APIs** using **Node.js**, **Express.js**, and related technologies.

This repository covers core API concepts including JSON handling, Axios requests, authentication, REST APIs, and a complete Secrets Project.

---

## 📂 Project Structure

### 1.0 Json
- Understanding JSON format
- Parsing and stringifying data
- Working with JSON in Node.js

### 2.0 Axios
- Making HTTP requests using Axios
- Handling API responses
- Error handling with try/catch
- Working with async/await

### 3.0 API Authentication
- API keys
- Basic Authentication
- Bearer Tokens
- Secure request handling

### 4.0 REST APIs
- REST architecture principles
- HTTP methods (GET, POST, PUT, PATCH, DELETE)
- Building RESTful routes with Express
- Testing APIs

### 5.0 Secrets Project
- Fetching data from an external API
- Rendering dynamic content using EJS
- Secure API communication
- Real-world API integration

## ⚙ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/APIs.git
```

2. Navigate into the project directory:

```bash
cd APIs
```

3. Install dependencies:

```bash
npm install
```

4. Start the server:

```bash
node index.js
```

Or (if using nodemon):

```bash
nodemon index.js
```
